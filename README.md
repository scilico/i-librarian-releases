# *I, Librarian Pro* Releases

*I, Librarian Pro* is a modern PDF document organizer for research groups and individuals. This repository provides
official binary releases and documentation for self-hosted installations.

Visit [i-librarian.net](https://i-librarian.net) to learn more.

## Key Features
  * A wide array of features tailored for private research groups.
  * Manage your PDF collection in one place; no client-side syncing required.
  * In-browser PDF viewer with annotation and note-taking support.
  * Faceted full-text search with detailed snippets.
  * AI-powered PDF/Project chat, summaries, and smart search capabilities.
  * Integrated search of Arxiv, Crossref, IEEE, NASA, PubMed, and PubMed Central, ScienceDirect, Scopus, and 100M+ world-wide patent database
  * PDF note summaries with export.
  * Audio note recording.
  * Detailed user activity logs.
  * OIDC and SAML2 SSO authentication.
  * Accurate OCR for over 90 languages.

## Self-hosting Benefits
  * Maintain full control over your data for privacy and compliance.
  * Create an unlimited number of libraries.
  * Both yearly and perpetual licenses are available.
  * Distributed as a fast, statically-compiled Go binary.
  * Easy installation using Docker containers or native Linux binaries.
